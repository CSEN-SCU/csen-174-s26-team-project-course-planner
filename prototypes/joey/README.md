# Joey Prototype: Transcript to SCU Course Suggestions

This prototype lets a user provide transcript information (text, JPG, PDF, or TXT) and get next-course suggestions powered by Gemini 2.5 Flash.

## Setup

1. Copy `.env.example` to `.env`
2. Add your Gemini API key to `GEMINI_API_KEY`
3. Install dependencies and run:

```bash
npm install
npm run build:majors-db
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- The backend sends multimodal transcript input directly to Gemini and asks for one concise paragraph with 2-4 suggested courses.
- The majors requirements database is generated by scraping the SCU Undergraduate Bulletin and compacting requirements into `data/majors.compact.json`.
- The backend includes this compact majors database in Gemini prompts for size-efficient context.
- Recommendations are generated by the backend endpoint `/api/suggest-courses`.
