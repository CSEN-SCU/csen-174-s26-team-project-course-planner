# Prototype Ideas

## Course History + Requirement Visibility

- Add an option to click courses already taken and view:
  - requirements fulfilled
  - requirements still missing
- Consider using the CSEN 4-year plan PDF as a clickable class map.

## Alternate Entry Without Transcript

- Support a no-upload path where users can talk directly to a chatbot and build a schedule conversationally.

## Planning Views

- Include next-quarter planning with calendar integration.
- Also include a 4-year planning list by quarter:
  - simple class list per quarter
  - no time-slot detail required for long-range view

## Core UI Tabs

- Keep an interface with:
  - AI tab
  - Calendar tab
  - Search tab

## Data Handling / Privacy

- After transcript upload, store only:
  - fulfilled requirements
  - needed requirements
- Do not store other transcript data after processing.
